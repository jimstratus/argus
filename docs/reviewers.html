Reviewers

View config.yaml

The registry has 15 reviewers. Three are dual-route (a direct provider API plus OpenRouter) and are reordered by the routing preference; the rest are single-route API reviewers or paid-CLI reviewers. Model versions are the headline of this release: GLM‑5.2, MiniMax M3, and the new default DeepSeek V4 Pro.

Dual-route vs CLI

The three dual-route reviewers respect route_preference. CLI reviewers (Codex / Claude / OpenCode / Gemini) keep their subscription primary with OpenRouter as a true fallback, no matter the routing preference. See Routing preference.

Ready-made groupings of these reviewers live in Profiles.
